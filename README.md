Driving e-paper screen using CircuitPython

## Hardware
* Raspberry Pi Pico W

* MH-ET Live 1.54 inch e-paper display

## Connections
| e-paper | Raspberry Pi |
| :-----: | :----------: |
| SCLK    | GP18         |
| SDI     | GP19         |
| CS      | GP17         |
| DC      | GP20         |
| Reset   | GP21         |
| Busy    | GP22         |
